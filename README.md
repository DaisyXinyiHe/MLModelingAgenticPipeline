## Pipeline Elements
* Data
  * Input data in CSV format
* Feature Engineer Agent
  * Take in data, problem statement, target variable, variable definition
  * Output feature engineering and data transformation suggestions
  * Write code based on the above suggestions and transform data
* Modeling Agent
  * Suggest Best modeling practice based on problem statement, target variable, and data
  * Write code based on the above suggestions and train model
* Evaluation Agent
  * Suggest modeling optimization based on training results
